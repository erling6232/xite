XITE Reference Manual

For version 3.44

The XITE Reference Manual provides access to all documented XITE executable programs and functions, including those which handle the BIFF image format and the ximage toolkit.

There is more information available about XITE from the Image Processing Laboratory in Department of Informatics, University of Oslo.
